# SkillGap Finder

## Vision

SkillGap Finder is an AI-powered career intelligence platform that helps students and job seekers understand how closely their current skills match their target job role.

Users can upload a resume and enter a target role.

The system analyzes the resume, identifies matching skills, missing skills, strengths, and weaknesses, and provides personalized recommendations to improve job readiness.

The goal is not to provide an ATS score, but to provide actionable career guidance through AI-powered analysis and mentoring.

## MVP Features

* Resume Upload
* Target Role Input
* Resume Analysis
* Skill Match Percentage
* Matching Skills
* Missing Skills
* Recommendations
* AI Mentor
